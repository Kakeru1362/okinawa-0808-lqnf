# 沖縄たび日程 🌺

沖縄旅行（2026.8.8–8.10）のスケジュールアプリ。スマホ専用。

- 日別タブ + 分類フィルタで予定を見やすく表示
- 分類（ご飯 / カフェ / アクティビティ / 移動 / 宿・休憩 / その他）で色分け、予約済みはバッジ表示
- 予定の追加・編集・削除、Googleマップのリンク、メモ
- Firebase Realtime Database でふたりのスマホ間をリアルタイム同期

宮古島たび日程と同じ機能で、見た目だけシーサー＆ハイビスカスのかわいい沖縄仕様。
